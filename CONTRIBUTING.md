# 贡献指南

欢迎贡献。这个仓库是 Gem1 流量工厂——老D 内容操盘系统的流量选题引擎。

## 什么值得贡献

- **新的选题代码**：如果你在实践中发现了一个反复有效的选题模式，欢迎提 PR 加新代码
- **案例补充**：用 Gem1 跑出了好结果？把你的输入+输出贡献到 `output-examples/`
- **代码适配**：把现有代码适配到新行业（餐饮/装修/教育/美业等）
- **Bug 修复**：SKILL.md 中的逻辑错误、触发词冲突、格式问题

## 什么不适合

- 纯理论讨论（去 Discussions 区）
- 与 Gem1 方法论无关的新功能
- 大幅改写核心方法论（请先开 Issue 讨论）

## 提交流程

1. Fork → 开分支
2. 改动 → 自测（把 SKILL.md 放入你的 Claude Code skills 目录跑一次）
3. PR 描述写清楚：改了什么、为什么改、测试结果
4. 中文为主，允许英文

## 行为准则

- 讨论方法论，不讨论人品
- 质疑观点欢迎，质疑动机不欢迎
- 老D 有最终裁定权（这是他的系统）
