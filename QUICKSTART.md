# 5 分钟上手

## 前提

- 已安装 [Claude Code](https://claude.ai/code)

## Step 1：安装（30 秒）

```bash
cp SKILL.md ~/.claude/skills/gem1-traffic-factory.md
```

## Step 2：说一句话（30 秒）

```
跑Gem1 我是做炒饭摊的。
```

## Step 3：回答问题（2 分钟）

Gem1 会问你几个问题——关于你的行业里真实发生的事。照实回答。元素越多越好。

## Step 4：看方向（1 分钟）

Gem1 输出 20 条方向。每条一行。你觉得能拍的就留着，觉得没意思的就跳过。

## Step 5：不满意就换一批

```
换一批
```

不同的盲抽 → 不同的方向。
