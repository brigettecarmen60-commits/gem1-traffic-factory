# 15 个人性按钮 · 详解

> 所有爆款内容的底层，都踩在这 15 个按钮上。不是发明——是拆了几千条爆款后统计出来的。
> 每个按钮 = 一个人性驱动力。选题时问自己：这条选题踩中了哪个按钮？

---

## 按钮全景

```
钱 ———— 贪婪/损失/占便宜。人对"获得"和"失去"钱的敏感度不对等。
安全 ——— 恐惧/隐私/身体/财产。威胁越具体越近身，触发越强。
公平 ——— 正义/双标/不公/被欺负。人对不公平的愤怒是生理性的。
真假 ——— 欺骗/伪装/内幕/真相。识别真假是生存本能。
对立 ——— 我们 vs 他们。制造阵营，站队即参与。
身份 ——— 我是谁/我想成为谁/我害怕成为谁。
隐私 ——— 窥探欲/被窥探的恐惧。带你看不该看的东西。
时间 ——— 遗憾/怀旧/紧迫/来不及。时间是不可再生资源。
极限 ——— 极端场景/极端行为。打破常规认知的边界。
死亡 ——— 终极恐惧/终极关怀。所有恐惧的底层。
遗憾 ——— 错过/如果/本可以。比失败更痛的是"差一点"。
争议 ——— 不认同/吵架/选边。中立不引发反应，站队才有互动。
未知 ——— 好奇/探索/揭秘。人对信息缺口有强迫性的填补冲动。
多巴胺 — 爽/解压/满足/惊喜。即时感官奖励。
性 ———— 吸引力/繁衍/荷尔蒙。最底层也最容易被滥用的按钮。
```

## 逐按钮拆解

### 1. 钱（贪婪/损失/占便宜）

**人性原理**：人对"获得"和"失去"钱的敏感度不对等——失去的痛苦是获得的 2 倍（损失厌恶）。所以这个按钮两条路都能走——告诉他能赚，或者告诉他正在亏。

**五种爆款格式**：

| 格式 | 逻辑 | 赛道适配 | 示例 |
|------|------|----------|------|
| 扒成本 | "你掏了钱，但不知道钱花在哪了。我告诉你。" | 餐饮/家居/娱乐 | 《一碗面38到底贵不贵——我把原材料成本全扒开了》 |
| 对比送检 | "你看到多花的冤枉钱到底花在哪了。" | 美妆/装修/教育 | 《把3000块大牌精华和35块国货同时送实验室——结果我沉默了》 |
| 避坑省钱 | "什么东西别在这买。买对了省好几万。" | 装修/母婴/宠物 | 《装修中最不能在实体店买的7样东西——都是暴利》 |
| 算总账 | "一笔一笔小钱，加成一个吓人的总数。" | 全赛道 | 《每天一杯奶茶——30年多少钱。算完我自己吓一跳》 |
| 揭秘暴利 | "你以为你在买X——其实你在买一堆Y。成本不到Z。" | 食品/美妆/保健品 | 《科技与狠活》 |

**关键原则**：大部分人不碰这个按钮，因为他们觉得"我的产品是什么"比"观众关心什么"重要。碰按钮的意思是——你说了一件事，跟你的产品没关系，但观众因为这件事停下来。停下来之后，他才会看到你的产品。

### 2. 安全（恐惧/隐私/身体/财产）

**人性原理**：安全需求是马斯洛第二层，仅高于生理需求。威胁越具体、越近身，触发越强。

**公式**：一个具体的人 + 正在做一件具体的事 + 如果不停止/不知道 + 会有具体的后果

**关联代码**：02末日清单 / 08叫停 / 09后悔药 / 10恐吓预警 / 40行业卧底

**关键原则**：不是恐吓。是告诉你——有件事你可能不知道，不知道的代价比知道大。前者可看可不看，后者不敢不看。

### 3. 公平（正义/双标/不公/被欺负）

**人性原理**：猴子实验证明——两只猴子做同样的事，一只拿黄瓜一只拿葡萄，拿黄瓜的那只直接把黄瓜摔了。人对不公平的愤怒是生理性的。

**公平选题的规律**：永远站一个"弱势方"，永远打一个"强势方"——大品牌、大公司、潜规则、专家权威。你不需要真的去打架，你只需要替他们说出来。

**关联代码**：16弑神 / 17解构 / 44共同敌人 / 48攻击权威 / 49群体共鸣 / 50阴阳怪气

### 4. 真假（欺骗/伪装/内幕/真相）

**人性原理**：人对被欺骗的恐惧根植于进化——被骗意味着资源损失、危险误判。识别真假是生存本能。

**关联代码**：16弑神 / 17解构 / 34阴谋论拼图 / 36钓鱼执法 / 40行业卧底 / 42八卦

### 5. 对立（我们 vs 他们）

**人性原理**：制造阵营，站队即参与。中立不引发反应，选边才有互动。

### 6. 身份（我是谁/我想成为谁/我害怕成为谁）

**人性原理**：人对"自我定义"的需求是核心驱动力。内容要么帮观众确认"我是谁"，要么帮观众想象"我可以成为谁"。

### 7. 隐私（窥探欲/被窥探的恐惧）

**人性原理**：人对他人隐私有天然好奇，对自己隐私有天然防备。两者都是内容驱动力。

### 8. 时间（遗憾/怀旧/紧迫/来不及）

**人性原理**：时间是唯一不可再生资源。和时间有关的情绪——遗憾、怀旧、紧迫——都有极高传播力。

### 9. 极限（极端场景/极端行为）

**人性原理**：极端打破常规认知，创造"不可不看"的视觉奇观。

### 10-15（简要）

| 按钮 | 核心驱动力 | 一句话 |
|------|-----------|--------|
| 死亡 | 终极恐惧 | 所有恐惧的底层，用得好是神作，用不好是消费悲剧 |
| 遗憾 | "差一点" | 比失败更痛的是差一点。时间+遗憾=核弹级情绪 |
| 争议 | 不认同/吵架 | 中立不引发反应，站队才有互动。立场即流量 |
| 未知 | 好奇心缺口 | 人对信息缺口有强迫性填补冲动。留白比填满更钩人 |
| 多巴胺 | 即时奖励 | 爽、解压、满足、惊喜。短视频的硬通货 |
| 性 | 吸引力 | 最底层也最容易被滥用。慎用 |

## 使用规则

1. **每条选题至少踩中 1 个按钮，理想情况是 2-3 个交叉。** 比如"理发店非让你办卡"——钱（被多收）+ 公平（被强迫）+ 安全（隐私被侵犯）。
2. **不要用按钮名称做选题。** "今天我们来聊聊公平"——这是自杀。按钮是后台引擎，选题是前台结果。
3. **同一个赛道，不同按钮切出完全不同的内容。** 做装修的——钱按钮 → 扒成本对比；安全按钮 → 甲醛检测揭秘；公平按钮 → 开发商精装房的坑。
4. **按钮是统计结果，不是理论推导。** 这 15 个按钮来自拆解上千条爆款后的归纳。如果哪天出现第 16 个，加进来就是了。
