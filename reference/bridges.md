# 桥段库 · 完整版

> 来源：老D武器库原始文件。不压缩、不缩写。
>
> **桥段定义：** 代码机制的"可执行翻译层"——把抽象代码翻译成具体可拍动作的承载方式。
>
> **桥段铁律（三必须）：**
> 1. 必须是某个代码的具体落地（不是独立的娱乐范式）
> 2. 必须商家一人/小团队能拍（不需要明星/化妆师/动物训练/巨额道具）
> 3. 必须真实可发生（不需要演技，不需要预演结果，让真实自然生长）
>
> **桥段不是综艺套路。** 桥段是代码的"使用说明书"。

## 桥段总池（已验证可执行）

### 🔍 暗访扮装类

**桥段 A1｜假扮客户暗访同行**
- 适用代码：Code 22 扮猪吃虎 / Code 32 突击检查 / Code 41 私生活窥探
- 操作要点：不预约直接进同行店，装作不懂行的客户，带卷尺/录音/手电等工具
- 适配赛道：几乎所有有同行的服务/零售赛道
- 真实可达性：✅ 高（一人+几十块预算）

**桥段 A2｜雇路人替我暗访**
- 适用代码：Code 32 突击检查 / Code 60 全网点菜
- 操作要点：花 100-300 块雇真陌生人，给他几个具体问题去同行店问
- 适配赛道：同行同质化严重的服务赛道（代账/装修/门窗/培训）
- 真实可达性：✅ 高
- 慎用：雇佣关系要明确，不能造假对话

**桥段 A3｜带礼物找陌生人换故事**
- 适用代码：Code 22 扮猪吃虎 / Code 41 私生活窥探
- 操作要点：带具体礼物（酒/烟/茶等），找特定状态的陌生人（看起来没人的小店等），公开换故事
- 适配赛道：任何有"目标群体可见"的赛道
- 真实可达性：✅ 高

## 桥段库扩展规则

**未来增加新桥段的标准：**
1. 必须明确挂在 1 个或多个具体代码上（不能是独立娱乐范式）
2. 必须验证过（用户/操盘手/或真实爆款里有过）
3. 必须能跨多个赛道使用（单赛道桥段意义不大）
4. 必须满足三铁律（代码具体落地 / 一人小团队能拍 / 不需演技）

**严禁加入的"伪桥段"：**
- ❌ 化妆师把人变老变年轻
- ❌ 训练动物当评委
- ❌ 请明星 / 网红 / 专业演员
- ❌ 凭空想象的戏剧场景
- ❌ 综艺级专业团队配合的桥段
