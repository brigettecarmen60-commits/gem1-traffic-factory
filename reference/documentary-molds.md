# 5 大纪实模具 · 深度解析

> **补充参考。** 不是核心引擎的一部分。如果你拿到方向后不知道"怎么拍"——回到这里。
> 废除所有戏剧化表演。5 个模具的共同特征：**不需要演员、不需要剧本、不需要刻意制造冲突。**

---

## 模具 1：第一视角 POV（主观代入）

**一句话**：让观众用你的眼睛看世界。

**为什么有效**：
- 剥夺观众的选择权——他只能看到你想让他看的东西
- 制造"我在现场"的沉浸感——比第三人称更容易触发共情
- 隐藏你的表情——观众自己脑补你的情绪，脑补的永远比演出来的强

**怎么拍**：胸前挂机/手机手持，拍你正在做的事。镜头就是你眼睛。

**适合什么**：硬核操作、第一现场、无干预工作流。比如：工厂流水线实操、上门维修过程、食材挑选现场。

**反面教材**：POV 不是"拿着手机晃"。每一步动作必须有信息量——观众在看一个他没见过或做不到的过程。如果他看 3 秒就知道后面会发生什么，他划走。

## 模具 2：桌面固定机位（物理检验）

**一句话**：把证据放在桌上，让镜头当法官。

**为什么有效**：
- 固定机位 = 没有人为干预 = 更可信
- 俯拍 = 上帝视角 = 观众觉得"我看穿了"
- 物品对比/拆解 = 眼见为实 = 不需要你说服他，他看到了

**怎么拍**：手机架上支架，垂直俯拍桌面。物品逐一入镜，对比、拆解、展示细节。

**适合什么**：实物对比、零件拆解、纸质证据展示、算账推演。

**反面教材**：桌面机位最怕的是"桌上什么都没有"。每次切换物品前，先想好：现在入镜的这件东西，观众能不能一眼看出信息量？如果没有，别放。

---

## 模具 3：第三人称手持跟拍（吃瓜旁观）

**一句话**：你就是路过看到了一件事，顺手拍了下来。

**为什么有效**：
- 晃动感/呼吸感 = 这不是摆拍 = 更真实
- 第三人称 = 你是旁观者不是主角 = 观众跟你站在同一侧
- "偷拍感" = 被拍的人没有表演意识 = 更可信

**怎么拍**：手持手机，跟在被拍对象后面或侧面。保持距离，不干预。

**适合什么**：旁观干活、突击现场、记录真实反应。

**反面教材**：跟拍不要变成"你拿着手机围着人家转"。最佳跟拍是——你站在一个固定位置，只转身体不挪脚步。晃动来自手腕，不是来自走位。

## 模具 4：监控/偷拍视角（窥私上帝）

**一句话**：让观众觉得"只有我看到了，别人都不知道"。

**为什么有效**：
- 画质粗糙 = 这不是演的 = 这是"真实发生的事"
- 固定广角 = 一览无余 = 观众获得"全知全能"的快感
- 被拍者不知道 = 无防备 = 真实行为

**怎么拍**：手机固定在角落/高处，广角，一镜到底不切。或者用真正的监控画面（须守法打码）。

**适合什么**：记录无防备的真实行为、时间流逝、空间全貌。

**反面教材**：监控视角的内容必须有一个"如果我不告诉你，你永远不会知道"的信息。如果拍到的内容跟正常生活一模一样，这就是一段无聊的监控录像。监控是形式，信息差才是内容。

---

## 模具 5：封闭空间单口（真实倾诉）

**一句话**：车里、深夜工位、路边摊——把门关上，说真话。

**为什么有效**：
- 封闭空间 = 安全感 = 说话的人更放松 = 更真实
- 固定机位 = 没有视觉干扰 = 观众的注意力全在你说的话上
- 夜晚/暗光 = 情绪放大器 = 更容易说出白天不会说的话

**怎么拍**：手机固定（方向盘支架/桌面支架），自然光或不打灯。像跟一个人聊天一样说话。卡壳了就卡了，不要重录。

**适合什么**：下班后的情绪发泄、行业吐槽、个人经历分享、深夜思考。

**反面教材**：封闭单口最怕"播音腔"。一旦你开始用"大家好，今天我们来聊聊"开头——这个模具就废了。它的全部力量在于：你在跟一个人说话，那个人刚好在镜头后面。

---

## 为什么是这五个？

这 5 个模具的共同设计原则：

1. **不需要演员**。你、你的同事、你的客户——真实的人做真实的事。
2. **不需要剧本**。有框架（拍什么/从哪拍/怎么拍），但没有台词。真实反应永远比写的台词好。
3. **不需要设备**。一台手机全搞定。模具的存在就是为了回答"手机怎么拍出不像手机拍的东西"。
4. **物理真实优先**。5 个模具都在解决同一个问题：怎么让观众相信这是真的。不是"拍得好看"——是"拍得真"。

**被砍掉的候选模具**：
- "多人对话/圆桌讨论" → 被砍。需要多人配合，普通人协调成本太高。
- "户外走动口播" → 被砍。环境噪音不可控，注意力分散，对新人不友好。
- "产品展示/开箱" → 被砍。已经烂大街，且容易滑向硬广。
