## 这是什么

Gem1 流量工厂是一个 Claude Code Skill。告诉它你是做什么的，它问你几个问题拿到元素，然后从三套引擎（61 代码 / 115 技法 / 桥段库）随机盲抽，碰撞出方向。

**它不做的事：** 不替你做创意判断。不编造行业细节。不替你决定哪条方向好。

**它做的事：** 碰撞。传播性预估。合规标记。

## 三步流程

| Step | 做什么 | 
|------|--------|
| 1. 拿元素 | 追问拿到真实的、圈外人能秒懂的行业元素 |
| 2. 碰撞 | 三引擎盲抽 × 元素 = 20 条方向 |
| 3. 标记 | 传播性预估 + 合规检查。不替用户决定删不删 |

---

## 双引擎

- **61 代码** — 心理机制。告诉你"观众为什么停"
- **桥段库** — 物理执行。告诉你"手机架在哪拍什么"

双引擎平等。各盲抽号码。抽到什么用什么。同一主题换皮超过 3 条强制重碰。
